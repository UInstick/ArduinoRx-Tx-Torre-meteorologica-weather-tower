# ArduinoRx-Tx-Torre-meteorologica-weather-tower
Proyecto de comunicacion a travez de modulos RF24, enviando datos recolectados por sensores analogicos
Este proyecto esta compuesto por los siguientes materiales.

2 Arduno Uno
2 Nrf24l021
1 Sensor de temperatura
1 Sensor de Humedad
1 Sensor de precipitacion 
1 Sensor de velocidad del viento
1 Shield Ethernet
Fuentes de alimentacion
Cable de red
Interfaz de usuario desarrollada en lenguaje de preferencia
